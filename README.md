# Kiro Sandbox

Development sandbox for testing and building projects with Kiro AI.

## Projects

### Monterey Bay Conditions Dashboard

A real-time ocean conditions dashboard for Monterey Bay, California. Provides live data for diving, fishing, kayaking, and surfing activities.

**Location:** `monterey-bay-dashboard/`

**Features:**
- Real-time NOAA tide, weather, wind, and swell data
- Dive conditions with visibility and surge risk
- Activity-specific recommendations (Go/Caution/No-Go)
- Species information with seasonal availability
- Firebase Analytics integration
- Deployed on Firebase Hosting

**Live Site:** [montereydashboard.web.app](https://montereydashboard.web.app)

**Tech Stack:**
- Next.js 16 with React 19
- TypeScript
- Tailwind CSS 4
- Firebase (Hosting + Analytics)
- NOAA APIs (CO-OPS, NDBC, NWS)
