# Components

This folder contains React components for the Monterey Bay Conditions Dashboard.

## Structure

- **Location Selector** - Allows users to switch between Monterey Bay spots
- **Activity Selector** - Allows users to choose: Boating, Fishing, Diving
- **Conditions Cards** - Display tide, marine weather, wind, swell, and dive conditions
- **Recommendation Card** - Shows Go/Caution/No-Go status with explanations
- **Forecast Panel** - Shows current and near-term forecast windows
- **Data Source Status** - Shows last updated time and data source information
