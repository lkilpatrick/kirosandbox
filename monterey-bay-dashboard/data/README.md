# Mock Data

This folder contains mock/sample data for development and testing.

## Data Files

- **locations.json** - Monterey Bay locations (Monterey Harbor, Breakwater, Stillwater Cove, Sand City, Moss Landing)
- **tide-data.json** - Sample tide predictions
- **weather-data.json** - Sample marine weather forecasts
- **wind-data.json** - Sample wind conditions
- **swell-data.json** - Sample swell forecasts
- **dive-conditions.json** - Sample dive condition estimates

## Purpose

Mock data enables:
- UI development without API dependencies
- Consistent testing scenarios
- Demonstration of various condition states (favorable, caution, no-go)
- Rapid prototyping of recommendation logic
